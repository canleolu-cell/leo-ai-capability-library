RECIPES / COPY & RUN

常用工作流

01

把对话变成知识条目

archive-chatgpt-conversations → Documents → Google Drive

归档这组对话，提炼决定、方法与待办，并整理进知识库。

02

从想法到发布网站

Sites → Imagegen → Netlify / Sites Hosting

把这份简报做成网站，完成视觉、内容、检查并发布。

03

把评审变成可合并改动

GitHub → gh-address-comments → gh-fix-ci

检查这个 PR 的未解决意见，修复选定问题并验证检查。
